Detailed Description

The Card class represents a playing card.

The Card class is the primary class of the CBJCards library. It represents a playing card with a suit, rank, and ace level. The ace level is only relevant if the card is an ace.

Additionally, cards can either be face up or face down.

◆ toNumericValue()

int CBJCards::Card::toNumericValue ( ) const

Returns the numeric value of the card.

The toNumericValue function returns the numeric value of the card. The numeric value of an ace is determined by the ace level of the card. The numeric value of a face card (Jack, Queen, King) is 10. The numeric value of a number card (2 through 10) is the same as the rank of the card.

Returns
The numeric value of the card.
